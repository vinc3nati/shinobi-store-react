<a name="features"></a>

## Features

- Sleek Landing Page
- Login and Sign up with [MockBee](http://mockbee.netlify.app/)
- Category filters
- Filter by price range, low to high, high to low
- Search for products
- Private Routes
- Cart Management
- Wishlist Management
- Payment Integration with [Razorpay](https://razorpay.com/)
- Order summary page
- User Profile Page
- Address Management
